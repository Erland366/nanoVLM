# TwinTower nanoVLM
## Setup
1. Install [uv](https://docs.astral.sh/uv/getting-started/installation/) package manager
2. Simply run `setup.sh`

## Directory
1. Training code -> `train_*.py` (vanilla / twin_tower)
2. TwinTower model -> `models/twin_tower.py`
3. Configs -> `configs/config_*.py` (vanilla / twin_tower)
4. Evaluation -> `generate.py` & `cider.py`
